h2. Keys

* A: @W@ (or @Z@ on QWERTY keyboards)
* B: @X@
* Start: @Enter@
* Select: @Backspace@
* Directional keys: Arrows

h2. How to compile (Linux and Mac OS-X only)

* Install @SFML@ (>= 2.x)
* Run @make@ at the root of the game folder
* Run @make run@, and enjoy!
